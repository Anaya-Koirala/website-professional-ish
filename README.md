##[KOIRALA.XYZ](https://koirala.xyz)

## Windows98 themed website now written in Flask.

- Support for blogs (writings) and a message board
- Multiple Windows98 apps now on the web (Courtesy of projects like [98.js.org](https://98.js.org/) and [JS Paint](https://jspaint.app))
- Special Layout for mobile phones
- Support for the old static site layout (Now a separate [branch](https://github.com/Anaya-Koirala/website-professional-ish/tree/old))
- Suggestions, fixes and everything in between encouraged and deeply appreciated.
