My fourth hour project for this class was the implementation of the Ultimate Tic Tac Toe game in Flutter.

The rules of the game are:

  • The game is played on a 3x3 grid of 3x3 tic-tac-toe boards.
  • Players take turns placing their mark (X or O) on the smaller boards.
  • The position of the mark on the smaller board determines which smaller board the next player must play on.
  • If a player is sent to a board that is already won or full, they can choose any board to play on.
  • The game is won by winning three smaller boards in a row on the larger grid.
